# RedaPlay
RedaPlay é uma plataforma gamificada para aprendizado de redação, desenvolvida durante o hackathon do programa Ganhe o Mundo, promovido pela Cesar School em parceria com a Embraer. Construída com Flask, SQLite e Python no back-end, e HTML, CSS e JavaScript no front-end, a aplicação oferece um sistema CRUD completo para gerenciamento de usuários, incluindo a modificação de e-mail e senha, além de lições em vídeo e o envio de redações. O diferencial fica por conta da mecânica de gamificação, onde cada acerto soma pontos e cada erro resulta na perda de vidas, transformando o estudo da produção textual em uma experiência dinâmica e envolvente.
